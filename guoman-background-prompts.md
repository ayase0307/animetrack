# 國漫玄幻背景圖提示詞

建議先生成橫幅背景，尺寸 1920x1080 或 2560x1440。生成後放到 `assets/`，再把 `anime-tracker-guoman-draft.html` 裡的：

```css
--fantasy-bg-image:none;
```

改成：

```css
--fantasy-bg-image:url("assets/你的檔名.png");
```

## 方向 A：冷墨玄域主背景

玄幻中國動畫風格的網站背景，深墨色夜空，遠處雲海與山脈剪影，青綠色靈氣光霧，少量金色符文線條漂浮，宣紙顆粒質感，畫面中央偏暗、四周有細緻光暈，適合 UI 文字疊加，沒有角色，沒有文字，沒有 logo，沒有建築特寫，高級感，電影級光影，wide composition, 16:9

## 方向 B：宗門玉簡介面背景

中國玄幻動畫 UI 背景，古籍玉簡與陣法靈紋的抽象視覺，深綠黑與溫金色調，半透明卷軸紋理，細密金線形成圓形法陣，留出中間大面積暗色空間給網頁內容，低對比，精緻但克制，沒有文字，沒有角色，沒有 logo，no readable text, 16:9

## 方向 C：水墨雲舟與星河

國漫水墨玄幻場景背景，夜色星河下的雲海，遠方有若隱若現的仙山和雲舟剪影，青玉色靈光穿過雲層，朱砂色只作極少量點綴，宣紙紋理，柔和景深，畫面不能太亮，適合放在追劇工具網站背景，沒有角色，沒有文字，沒有 logo，16:9

## 負面提示詞

不要人物，不要臉，不要巨大的中文文字，不要 logo，不要現代城市，不要科幻機甲，不要過亮霓虹，不要紫藍賽博龐克，不要雜亂細節，不要佔滿畫面的主體，不要低解析，不要模糊 UI 文字。

# Logo 生成提示詞

建議生成透明背景 PNG 或 SVG 風格圖，橫向比例約 3:1 或 4:1。Logo 要能放在深墨色網站 header 中，線條清楚，不要太多細節。生成後可命名為 `zhuiju-helper-logo-guoman.png` 或 `zhuiju-helper-logo-guoman.svg`。

## 推薦方向：冷墨玉印字標

為網站「追劇小幫手」設計一個可放在深墨色 header 中的橫向 logo，透明背景，繁體中文「追劇小幫手」清楚可讀，主圖形是一枚簡化的青玉印章，印章內藏播放鍵與一筆墨雲，整體是國漫玄幻動畫 UI 風格但保持產品工具感。配色使用深墨、青玉、溫金細線、極少量朱砂印記；線條要少、輪廓要乾淨，小尺寸仍清楚。不要角色，不要寫實龍，不要複雜法陣，不要大面積發光，不要背景場景，不要英文，不要簡體字。風格關鍵詞：冷墨、水墨雲氣、青玉印、金箔細線、朱砂小印、現代化宋體與篆刻混合、乾淨可向量化。

## 方向 B：墨龍雲紋字標
